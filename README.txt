1. Run the 'index.html' file on any browser directly.
2. Or use 'XAMPP' to run the application.
	a. If you use 'XAMPP' then keep the main folder within the htdocs folder under 'XAMPP'.
	b. Now open any browser and type 'http://localhost/NUMA/' on address bar and run the application.


3. Application is responsive. Add new record, update record, progress bar etc. functionalities are there.
